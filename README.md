# nodejs_study
모던 웹을 위한 Node.js 프로그래밍 예제


### 추가할 기능
  #### geolocation
    - 가입 로그인 기능
    - Tracker 페이지에서 지도 확인
  #### shopping_cart
    - 로그인 기능
    - 카트에 여러개의 물건을 넣고 한번에 구매
    - 카트에서 빠져나오는 시간을 실시간으로 표시
  #### canvas
    - 그림 저장
    - 저장한 그림 불러오기
    - 채팅방 추가
    - 지우개 추가
    - 터치 이벤트 추가로 터치로 그림 그리기
